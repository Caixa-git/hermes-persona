# CDPD 수학적 증명

## 최적 알고리즘

```
optimal(S) = {
    SINGLE_PERSONA  if S < 2   (ΔU < 0)
    MULTI_PERSONA   if S ≥ 2   (ΔU > 0, minor_count = min(S, 3))
}
```

## 효용 함수

U(S) = Q(S) − η·C(S)/k

η 기본값 0.3에서:

**S=0:** ΔU = (0.90−0.85) − 0.3·(55,886−12,764)/12,764 = 0.05 − 1.01 = −0.96
**S=2:** ΔU = (0.95−0.60) − 0.3·(74,472−162,831)/12,764 = 0.35 + 2.08 = +2.43

## 임계값

T ≈ 1.18, ceil(T) = 2

S=1과 S=2 사이 ΔU 차이 = 0.82 — 노이즈에 흔들리지 않는 차이.

## 저지연·고품질 보정

ψ↑ → η×1.2 (비용 민감 증가)
Ω↑ → η×0.8 (비용 무시)

η' = η × 1.2 × 0.8 ≈ η (상쇄되어 T 유지)

## 모델 교정

4개 실험으로 Φ_s, Φ_m, α, β, Γ_base, k 재측정 후 T 재계산.
